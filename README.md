# 🎬 Movie Explorer

A responsive movie search app using OMDb API. Search for any movie, view details, and save favorites to localStorage.

## Features
- Search movies by title
- Responsive grid layout (CSS Grid + Flexbox)
- Modal with full plot, rating, cast
- Add/remove favorites (persisted)
- Loading states & error handling

## Technologies
- HTML5 (semantic)
- CSS3 (grid, flex, animations, media queries)
- JavaScript ES6 (async/await, fetch, localStorage, event delegation)
